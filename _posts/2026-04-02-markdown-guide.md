Welcome to your new Jekyll-powered blog architecture! You can now write inside `.md` files without ever touching HTML again. Everything perfectly hooks into your background physics canvas natively.

## Titles and Headings
To create a section title, use hashtags. The more hashtags you add, the smaller the heading becomes:
### This is an H3
#### This is an H4

## Code Blocks
To insert a code block with the *Atom Dark* or *Atom Light* syntax highlighting seamlessly managed by your theme toggle, use triple backticks and specify the language format:

```javascript
// This will render with beautiful colors
function greet() {
    console.log("Hello, world!");
}
```

## Highlight Boxes (Callouts)
To create a stylized highlight or quotation box to draw a reader's attention to an important concept, use the `>` arrow symbol to open a blockquote:

> **Important:** This is a highlight box! It is perfect for critical quotes, callouts, or summaries that you want to stand out from the rest of the text.

## Lists
- You can create bullet points
- Just by starting lines with dashes
  - And indenting them makes sub-lists!

## Emphasis and Links
You can make your text **bold** using double asterisks and *italic* using single asterisks.
You can also [create invisible hyperlinks](https://apple.com) directly using brackets around your text followed by the URL in parenthesis.
